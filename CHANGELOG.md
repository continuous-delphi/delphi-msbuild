## [0.1.0] - 2026-03-16

- Initial release of `delphi-msbuild.ps1` -- build Delphi `.dproj` projects
  via MSBuild from the command line, with support for piped output from
  `delphi-inspect` and automatic `rsvars.bat` environment sourcing.
